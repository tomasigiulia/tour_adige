=====================================
   TOUR VIRTUALE - ISTRUZIONI D'USO
=====================================

COME AVVIARE IL TOUR VIRTUALE:
-------------------------------

1. Fare doppio clic sul file "Avvia_Tour_Virtuale.bat"

2. Si aprirà una finestra nera (prompt dei comandi) - NON CHIUDERLA!

3. Il browser si aprirà automaticamente con il tour virtuale

4. Navigare nel tour utilizzando il mouse e le frecce


COME CHIUDERE IL TOUR:
-----------------------

- Chiudere la finestra nera (prompt dei comandi)
- Oppure premere un tasto qualsiasi nella finestra nera


NOTE IMPORTANTI:
----------------

✓ Il file .bat deve rimanere nella stessa cartella dei file del tour

✓ Non spostare o eliminare i file nella cartella "panorami" e "img"

✓ La finestra nera DEVE rimanere aperta durante l'utilizzo del tour

✓ Se il browser non si apre automaticamente, aprirlo manualmente 
  e digitare: http://localhost:8000


REQUISITI:
----------

- Sistema operativo: Windows 7 o superiore
- Browser: Chrome, Firefox, Edge o Safari aggiornato
- Non richiede connessione internet
- Non richiede installazione


RISOLUZIONE PROBLEMI:
---------------------

Q: Il browser non si apre
A: Aprire manualmente il browser e andare su http://localhost:8000

Q: Appare un errore di sicurezza
A: Cliccare su "Esegui comunque" o "Consenti"

Q: Il tour non si carica
A: Verificare che tutti i file siano nella stessa cartella


Per assistenza, contattare lo sviluppatore.
